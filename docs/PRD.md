## 1. 제품 개요

이메일/비밀번호 인증 기반의 Todo + 건강 루틴 관리 웹앱.  
날짜별로 10개의 고정 일일 루틴과 자유 할일을 관리하고, 월간 달성률을 5개 카테고리로 리뷰할 수 있다.  
하루의 감정, 메모, 대표 사진을 함께 기록할 수 있다.

---

## 2. 목표

- 매일 반복되는 건강 루틴을 쉽게 체크
- 날짜별 자유 할일을 추가·관리
- 한 달 흐름을 카테고리별로 한눈에 리뷰
- 하루를 감정·메모·사진으로 기록
- 새로고침 후에도 데이터 유지, 사용자별 격리

## 4. 기능 요구사항

### 4.1 인증 (US-01~03)
- 이메일 형식 검증, 비밀번호 최소 6자
- 이미 사용 중인 이메일은 가입 불가
- 가입 즉시 로그인 전환, 새로고침 후 세션 유지

### 4.2 달력 (US-04~06)
- 로그인 직후 월간 달력이 기본 화면으로 표시
- 이전·다음 월 이동 가능
- 각 날짜 셀에 루틴 달성 프로그레스 바 + `N/10` 표시
- 내 할일이 있는 날짜에는 `+완료/전체` 뱃지 추가 표시
- 모든 루틴·할일 완료 시 날짜 셀 초록색 강조

### 4.3 내 할일 (US-07~12)
- 화면 최상단에 배치 (보라색 좌측 강조선)
- 텍스트 입력 후 Enter 또는 추가 버튼으로 생성, 최신순 정렬
- 완료 항목은 취소선·초록 배경으로 표시
- 인라인 편집: ✎ 클릭 → Enter 저장 / ESC 취소
- 완료 항목 존재 시 "완료 항목 삭제" 버튼 표시

### 4.4 일일 루틴 (US-13~16)

**10개 고정 루틴:**

| # | 이름 | 카테고리 |
|---|------|----------|
| 1 | 인바디 측정 | 건강측정 |
| 2 | 공복혈당 측정 | 건강측정 |
| 3 | 단백질 95g 섭취 | 식단 |
| 4 | 애플워치 움직이기 링 완성 | 생활습관 |
| 5 | 하루 물 2L 마시기 | 식단 |
| 6 | 영양제 챙겨 먹기 | 식단 |
| 7 | 오늘 하루 리뷰하기 | 하루정리 |
| 8 | F45출석 | 운동 |
| 9 | RUN | 운동 |
| 10 | 그외운동 | 운동 |

- 카테고리별로 구분, 각 카테고리 내에서 2열 그리드 표시
- 체크 시 초록 체크 버튼 + 배경 전환
- 날짜별 독립 저장 (`habits_${userId}_${dateStr}`)

### 4.5 한달 리뷰 (US-17~20)

**5개 카테고리 색상:**

| 카테고리 | 색상 | 포함 루틴 |
|----------|------|----------|
| 건강측정 | 파란색 `#3b82f6` | 1, 2 |
| 식단 | 초록색 `#22c55e` | 3, 5, 6 |
| 생활습관 | 보라색 `#a855f7` | 4 |
| 운동 | 주황색 `#f97316` | 8, 9, 10 |
| 하루정리 | 민트색 `#14b8a6` | 7 |

- 각 카테고리: 미니 달력 그리드(일~토 7열) + 달성률 바 + %(완료일/경과일)
- 셀 색상: 미완료=투명, 부분 완료=연한 색, 전체 완료=진한 색, 미래=점선 투명
- 이전·다음 월 이동 가능

### 4.6 하루 기록 (US-21~23)

**감정 이모지 18개:**  
😊 🥰 🤩 😎 🥳 😄 🤗 😌 🙂 😴 😪 🥺 😢 😭 😅 😤 😡 🤔  
- 선택/재클릭으로 해제, 1개만 선택 가능
- 저장키: `emoji_${userId}_${dateStr}`

**하루 메모:**  
- 자유 텍스트, 자동 저장
- 저장키: `memo_${userId}_${dateStr}`

**오늘의 사진:**  
- 이미지 파일 업로드, 최대 900px 리사이즈 후 JPEG 0.72 품질로 압축
- base64로 localStorage 저장
- 저장키: `photo_${userId}_${dateStr}`

- 서버 / 외부 DB 연동
- OAuth / 소셜 로그인
- 비밀번호 찾기 / 재설정

- 루틴 항목 커스터마이징
- 푸시 알림
- 친구 공유 / 소셜 기능

## 8. 보안 고려사항

> **MVP 한계**: 비밀번호는 `btoa()` 인코딩 후 localStorage에 저장됩니다.  
> 실제 서비스 배포 시 백엔드 인증(JWT, OAuth)과 서버사이드 해싱으로 교체 필요.  
> 사진 데이터는 base64로 localStorage에 저장되며 브라우저 Storage 용량 제한(5~10MB) 적용.

## 9. 성공 지표

- 로그인 → 달력 → 날짜 선택 → 루틴 체크 → 내 할일 추가 전체 흐름 오류 없이 작동
- 한달 리뷰에서 5개 카테고리 히트맵 정확히 표시
- 감정·메모·사진 저장 후 날짜 이동 → 복귀 시 데이터 복원
- 새로고침 후 로그인 세션 및 모든 날짜 데이터 복원
- 모바일 뷰포트(375px)에서 UI 깨짐 없음
