# 감정일기 앱 — Product Requirements Document

**버전**: 1.0  
**작성일**: 2026-05-23  
**상태**: Draft

---

## 1. 제품 개요

매일의 감정을 귀여운 아이콘으로 기록하고, 사진과 텍스트로 나만의 이야기를 남기는 감성 일기 웹앱.  
친구와 일기를 공유하고 소통하는 SNS형 일기장.

---

## 2. 목표

- 감정을 직관적인 아이콘으로 빠르게 기록
- 사진과 손글씨 느낌의 텍스트로 감성적인 일기 작성
- 나만의 공간 + 친구와의 소통 공간 분리
- 새로고침 후에도 데이터 유지 (MVP: localStorage → 이후 클라우드)

---

## 3. 사용자 스토리

| ID | 스토리 |
|----|--------|
| US-01 | 사용자는 이메일/비밀번호로 회원가입 및 로그인할 수 있다 |
| US-02 | 사용자는 오늘의 감정을 귀여운 아이콘으로 선택할 수 있다 |
| US-03 | 사용자는 텍스트로 일기를 작성할 수 있다 |
| US-04 | 사용자는 일기에 사진을 최대 3장 첨부할 수 있다 |
| US-05 | 사용자는 일기 목록을 최신순으로 볼 수 있다 |
| US-06 | 사용자는 일기를 수정하고 삭제할 수 있다 |
| US-07 | 사용자는 특정 날짜의 일기를 달력에서 확인할 수 있다 |
| US-08 | 사용자는 친구에게 일기를 공유할 수 있다 *(v2)* |

## 4. 기능 요구사항

### 4.1 인증 (US-01)
- 이메일 + 비밀번호 회원가입 / 로그인
- 로그인 세션 유지 (새로고침 후에도 유지)
- 로그아웃

### 4.2 감정 선택 (US-02)

| 아이콘 | 감정 | 아이콘 | 감정 |
|--------|------|--------|------|
| 😊 | 행복 | 😢 | 슬픔 |
| 😡 | 화남 | 😴 | 피곤 |
| 🥰 | 설렘 | 😤 | 짜증 |
| 😮 | 놀람 | 🥲 | 뭉클 |

- 일기 작성 시 감정 1개 필수 선택
- 선택된 감정은 시각적으로 강조

### 4.3 일기 작성 (US-03, US-04)
- 날짜 자동 기입 (오늘 날짜 기본값, 변경 가능)
- 손글씨 느낌 폰트 (Nanum Pen Script 또는 Gaegu)
- 텍스트 최소 1자 이상
- 사진 첨부 최대 3장 (파일 선택 → base64 인코딩 저장)
- 작성 중 임시 저장 (localStorage draft)

### 4.4 일기 목록 (US-05)
- 최신순 정렬
- 각 항목: 날짜 + 감정 아이콘 + 텍스트 미리보기(50자) + 대표 사진 썸네일
- 무한 스크롤 또는 페이지네이션 *(MVP: 전체 표시)*

### 4.5 일기 수정 / 삭제 (US-06)
- 각 일기 상세 화면에서 수정 / 삭제 버튼 제공
- 삭제 시 확인 다이얼로그 표시

### 4.6 달력 뷰 (US-07)
- 월간 달력에서 일기 작성 날짜에 감정 아이콘 표시
- 날짜 클릭 → 해당 날짜의 일기 상세 보기 또는 작성

### 4.7 친구 공유 (US-08, US-09) — v2
- 공유 링크 생성 (해시 기반)
- 친구 초대 후 특정 일기 공유
- 공유된 일기에 이모지 리액션 + 댓글

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 데이터 지속성 | localStorage (MVP) → Supabase/Firebase (v2) |
| 사진 저장 | base64 인코딩 (MVP 한계: 용량 제한) → 클라우드 스토리지 (v2) |
| 반응형 | 모바일(375px~) 및 데스크탑 지원 |
| 폰트 | Google Fonts — Nanum Pen Script (손글씨), Pretendard (UI) |
| 감성 UI | 파스텔 컬러 팔레트, 부드러운 모서리, 귀여운 일러스트 포인트 |

---

## 6. 화면 구성 (IA)

```
앱
├── 로그인 / 회원가입
└── 메인 (로그인 후)
    ├── 달력 뷰 (기본 화면)
    │   └── 날짜 클릭 → 일기 상세 or 작성
    ├── 일기 목록
    ├── 일기 작성
    │   ├── 날짜 선택
    │   ├── 감정 아이콘 선택
    │   ├── 텍스트 입력 (손글씨 폰트)
    │   └── 사진 첨부 (최대 3장)
    ├── 일기 상세 / 수정 / 삭제
    └── 설정 (프로필, 로그아웃)
```

## 7. 기술 스택

| 레이어 | MVP | v2 |
|--------|-----|----|
| 프레임워크 | React 18 + Vite | 동일 |
| 스타일 | CSS (CSS Variables + 파스텔 팔레트) | Tailwind CSS 고려 |
| 폰트 | Google Fonts (Nanum Pen Script) | 동일 |
| 인증 | localStorage 기반 | Firebase Auth / Supabase |
| 데이터 | localStorage | Supabase PostgreSQL |
| 사진 | base64 localStorage | Supabase Storage / S3 |
| 배포 | Vercel / Netlify | 동일 |

## 8. 범위 외 (Out of Scope for MVP)

- 친구 초대 / 공유 / 댓글 (백엔드 필요)
- 클라우드 사진 업로드
- 푸시 알림 / 리마인더
- 해시태그 / 검색
- 다크모드
- 음성 일기

---

## 9. 디자인 방향

- **컬러**: 파스텔 핑크, 라벤더, 민트, 피치 계열
- **폰트**: 손글씨 느낌 (Nanum Pen Script) + 가독성 UI 폰트 (Pretendard)
- **감정 아이콘**: 크고 선명한 이모지, 선택 시 애니메이션 (scale up)
- **일기 카드**: 종이 질감 느낌, 살짝 기울어진 테이프 포인트
- **전반적 톤**: 귀엽고 따뜻한 다이어리 감성

---

## 10. 성공 지표 (MVP)

- 일기 CRUD 전체 흐름 오류 없이 동작
- 감정 아이콘 선택 → 목록/달력에 반영
- 사진 첨부 → 상세 화면에 표시
- 새로고침 후 모든 데이터 복원
- 모바일 375px 뷰포트에서 UI 정상 표시
